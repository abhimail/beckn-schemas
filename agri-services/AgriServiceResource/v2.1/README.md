# AgriServiceResource — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Extension of ServiceResource for agriculture-specific use cases. Adds livestock-specific fields, service categories tailored to agriculture, and hierarchical geographic coverage at the village/ward/county level.

### Container

- **Resource.resourceAttributes**

### Key features

- Agriculture service categories (vaccination, brooding, advisory, etc)
- Provider types (vet, extension officer, agrovet, etc)
- Livestock type specifications
- Hierarchical geographic coverage (village, ward, county)
- Extends all base ServiceResource fields
