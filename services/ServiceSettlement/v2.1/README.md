# ServiceSettlement — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Initial release of the ServiceSettlement schema targeting the v2.1 generalised model. This schema records payment settlement details including whether the provider has received payment, when it was recorded, and the receipt reference.

### Container

- **Settlement.settlementAttributes**

### Key features

- Payment receipt confirmation
- Payment recording timestamp
- Receipt reference tracking
- Settlement notes
